# Git-Pull-Intro
A repo to make your first PR in.

# How to Make Your First Pull Request
## Steps:
* 1. Fork the project in your Github Profile
* 2. Edit the contribution.md file and add your name in it
* 3. Save the changes 
* 4. Now go to the Pull Request tab and Click on Create New Pull Request option
* 5. Check the topic of the pull request and submit it.

## Hoooray!!! You just made your first successful Pull Request
